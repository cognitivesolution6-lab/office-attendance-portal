# 🏢 SmartAttend — Office Attendance Portal

A complete, ready-to-use **Attendance Management System** for offices — no server, no database, no installation required!

## ✨ Features

### 👤 Employee Features
- ✅ **Daily Attendance** — Check In / Check Out with one click
- 📍 **GPS Location** — Attendance ke saath exact location record hoti hai
- 🌴 **Leave Request** — Sick, Casual, Earned, WFH leave apply karo
- ✏️ **Daily Activity** — Apni daily activities log karo
- 📅 **Monthly Attendance** — Pura mahine ki attendance dekho
- 💸 **Salary Deductions** — Absent aur late ka deduction automatically calculate hota hai

### 🔑 Admin Features
- 👥 **Employee Management** — Employees add/delete karo
- 📋 **Today's Attendance** — Sab employees ki aaj ki attendance ek jagah
- 🌴 **Leave Approvals** — Pending leaves approve ya reject karo
- 📊 **Monthly Reports** — Complete salary deduction reports
- 📍 **Location View** — Har employee ki check-in location Google Maps pe dekho

## 📊 Deduction Rules
| Rule | Deduction |
|------|-----------|
| Absent | 1 din ki salary |
| Late (after 9:00 AM) | 1% monthly salary |
| Half Day | Half din ki salary |

## 🚀 Quick Start

### Option 1: Direct Use
Bas `index.html` file download karo aur browser mein open karo!

### Option 2: GitHub Pages (Free Hosting)
1. GitHub par **new repository** banao
2. `index.html` file upload karo
3. Settings → Pages → Source: `main branch` select karo
4. Done! Tumhara portal live ho gaya ✅
